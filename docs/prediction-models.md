# Prediction Models

## Uniform Swing Model
Applies a uniform national swing across all constituencies.
- Simple but ignores regional variation.
- Useful as a baseline.

## Regression Model
Uses demographic and historical data to predict constituency-level outcomes.
- Features: turnout, incumbency, region, previous vote shares.
- Target: winner party.

## Bayesian Structural Model
Incorporates polling data with prior election results using Bayesian methods.
- Produces probabilistic predictions (e.g. "Labour wins 65% chance").
- Accounts for polling error and house effects.
